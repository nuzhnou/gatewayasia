# Gateway Asia — план запуска (Вьетнам / Азия)

> **Бренд:** Gateway Asia · **бот:** `@gatewayasia_bot`
> **Источник вакансий на старте:** ручной каталог (скрейпинг — позже).
> Копия польского бота (@svoyakplBot), переориентированная на рынок Азии.
> ЦА кандидата: люди с европейской («белой») внешностью, английский язык, для работы
> учителями английского, хостес, моделями, работниками отелей и индустрии гостеприимства —
> везде, где западная внешность поднимает статус заведения.
> Языки общения бота: RU + EN (польский был только RU).

---

## ⚠️ Важно про нейминг и Facebook/рекламу

Бренд НЕ должен в открытую продавать «белую внешность» — Facebook/Meta и Google банят
формулировки с дискриминацией по внешности/расе. Поэтому:
- **Бренд = нейтральный, профессиональный** («европейские/западные специалисты в Азии»).
- **Критерий внешности = в таргетинге и скрипте диалога**, не в названии и не в публичных постах.
- Публично позиционируем как: *English teachers & hospitality talent — Western/European professionals for Asia.*

|-----|----------|
| Рабочее название | **(выбрать — см. блок «Нейминг» ниже)** |

## 2. Внутренние каналы управления (приватные, 5 шт)

Точная копия того, что уже есть в польском боте — просто заводим НОВЫЕ приватные каналы/группы
под Азию и подставляем их ID в `.env`. Бот сам туда пишет.

| # | Канал | Назначение | env-переменная |
|---|-------|-----------|----------------|
| 1 | 🔥 Hot Leads — Asia | горячие квалифицированные лиды (кандидат готов) | `ADMIN_ALERT_CHAT_ID` |
| 2 | 💬 Dialog Monitor — Asia | живая лента диалогов: 🆕 старт / ❄️ заглох / ✅ квалифицирован / 🚩 не справился | `DIALOG_MONITOR_CHAT_ID` |
| 3 | 📊 Daily Reports — Asia | ежедневный отчёт (лиды, выплаты, источники) в 21:00 | `DAILY_REPORT_CHAT_ID` |
| 4 | ⚙️ Lead Operations — Asia | служебные операции по лидам | `LEAD_OPERATIONS_CHAT_ID` |
| 5 | 🧪 Source Tests — Asia | тесты источников (новые FB-группы / TG-каналы) | `SOURCE_TESTS_CHAT_ID` |

> Это управляющий контур. Кандидаты сюда не заходят.

## 3. Публичные каналы для постинга вакансий (НОВОЕ — в польском не делали)

Это публичные витрины: туда автопостятся вакансии, кандидаты листают и жмут «написать боту».
Канал = бесплатный, вечный трафик (в отличие от FB-групп, где банят).

Рекомендую начать с **1 основного + 2 нишевых** (нишевые поднимают конверсию и выглядят экспертно):

| # | Канал | Контент | Кому |
|---|-------|---------|------|
| 1 | 🌏 Jobs in Asia (основной) | все вакансии Азии, RU+EN | широкая ЦА |
| 2 | 🎓 Teach English in Asia | только преподавание английского | учителя — самый массовый сегмент |
| 3 | ✨ Hospitality & Model Gigs Asia | хостес, модели, отели, ивенты, промо | «face value» сегмент |

Каждый пост в публичном канале ведёт CTA → в бота (`@<brand>_bot`).
Позже можно разбить по странам (🇻🇳 Vietnam Jobs, 🇹🇭 Thailand Jobs, 🇨🇳 China Jobs) — когда пойдёт объём.

## 4. Типы вакансий для Азии (заменяют польские вакансии)

Основа диалога и каталога. Где ценится западная/европейская внешность:

1. **Преподаватель английского** (ESL) — школы, садики, языковые центры. Массовый, легальный, визы.
2. **Хостес / гостевой менеджмент** — рестораны, клубы, лаунжи, открытия.
3. **Модель / промо** — фотосессии, показы, реклама, ивенты, шоурумы.
4. **Работник отеля / ресепшн / гость-сервис** — премиум-отели, курорты.
5. **MC / ведущий / event talent** — конференции, презентации, корпоративы.
6. **«Business presence» / представительские роли** — встречи, переговоры (имиджевая роль).

> Хранить как структурированные поля + SQL-фильтр (как договорились по архитектуре вакансий польского
> бота — НЕ RAG), чтобы боту подавать только подходящее подмножество.

## 5. Инфраструктура (что развести отдельно от польского)

Чтобы две системы не пересекались:

- [ ] **Отдельная папка** `asia-bot/` — ✅ создана (форкаем сюда код из `src/`).
- [ ] **Отдельный git-репозиторий** (или ветка) — напр. `asiabot`.
- [ ] **Отдельная БД Turso** — новый `TURSO_DATABASE_URL` (лиды Азии отдельно от Польши).
- [ ] **Отдельный сервис на Render** — свой `render.yaml`, свой keepalive.
- [ ] **Отдельный `.env`** — новый бот-токен, новые ID каналов, тот же стек Gemini-ключей (можно общий пул).
- [ ] **Свой список источников** — FB-группы и TG-каналы про работу в Азии для экспатов
      (аналог `data/facebook_sources.csv` и `config.json → sources`).
- [ ] **Локализация диалога** — добавить EN-ветку (польский был RU-only).
