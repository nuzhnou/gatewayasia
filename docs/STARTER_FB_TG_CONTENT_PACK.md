# Starter FB/TG Content Pack

Use this pack for the first 7 days of source testing. Every post must use a unique source code in the bot link:

`https://t.me/<BOT_USERNAME>?start=<SOURCE_CODE>`

Replace `<BOT_USERNAME>` with the real bot username from `.env`.

## Source Code Map

Start with 30 source codes:

| City / segment | Source codes |
| --- | --- |
| Warsaw | `fb_warsaw_001` to `fb_warsaw_005` |
| Wroclaw | `fb_wroclaw_001` to `fb_wroclaw_005` |
| Poznan | `fb_poznan_001` to `fb_poznan_005` |
| Gdansk | `fb_gdansk_001` to `fb_gdansk_005` |
| Lodz | `fb_lodz_001` to `fb_lodz_005` |
| General Poland groups | `fb_poland_001` to `fb_poland_005` |

Rules:

- One source code per group/post batch.
- Do not reuse source codes across different groups during the first test week.
- If a group produces 3+ bot starts, keep testing it with a new post and new source code.
- If a group produces zero starts after 3 attempts, stop using it for now.

## Daily Posting Plan

First 3 days:

- 10 groups per day.
- 1 post per group.
- 5-10 comments under relevant "looking for work" posts.
- Manual replies to every comment within 30 minutes if possible.

Days 4-7:

- Repeat top-performing groups.
- Add 10-20 new groups.
- Keep daily volume below account-risk levels.
- Scale comments before scaling identical posts.

## Offer Angles

Use five angles:

1. Work without Polish.
2. Work with housing.
3. Work for women.
4. Work for men/couples.
5. Fast start for people already in Poland.
